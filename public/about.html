About RunZip

RunZip is a tool for sharing static web projects instantly. Upload a ZIP file and get a live URL and QR code — no account needed, no deploy pipeline, no config.

How it works

01

Upload your project

Drop a ZIP containing your HTML, CSS, and JavaScript files.

02

Get a live URL

RunZip detects your entry file and hosts it automatically.

03

Share anywhere

Copy the link or scan the QR code from any phone.

Features

Instant preview

Test your project in desktop, tablet, and phone views right in the browser.

QR code sharing

Every project gets a scannable QR code for quick mobile access.

Smart detection

Automatically finds the right entry HTML file, even in nested folders.

Auto-cleanup

Projects are automatically removed after 10 minutes to keep things clean.
