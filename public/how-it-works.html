How to use RunZip

Three steps to share any static web project with a live URL and QR code.

Step by step

01

Select your ZIP file

Choose a ZIP archive that contains your static website or app files (HTML, CSS, JS).

02

Upload to RunZip

Drop the ZIP into the upload area. RunZip extracts the files and detects the entry page automatically.

03

Share the link or QR code

Copy the generated URL or scan the QR code from any phone to open the project.

Supported files

  • ZIP archives containing static project files
  • HTML, CSS, JavaScript, images, and other web assets
  • Nested folders are supported — RunZip finds the index.html automatically
  • Maximum upload size: 10 MB per ZIP file

QR sharing

  • After upload, RunZip generates a unique live link for your project
  • A QR code is created from that link
  • Anyone can scan it with a phone camera and open your project instantly
  • No app install or account required

FAQ

Is RunZip free?

Yes, completely free.

Do I need an account?

No. No login or signup required.

How long are projects hosted?

Projects are automatically cleaned up 24 hours after upload.

What's the file size limit?

Each upload supports ZIP files up to 10 MB.

Does it work on mobile?

Yes. Open the link or scan the QR code from any phone or tablet.
