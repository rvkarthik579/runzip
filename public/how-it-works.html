How to Use RunZip

RunZip helps you upload a ZIP file and instantly share your static web project with a live URL and QR code.

Step-by-step guide

01

Select your ZIP file

Choose a ZIP that contains your static website or app files.

02

Upload to RunZip

Drop the ZIP into the upload area and click upload. RunZip detects the entry page automatically.

03

Get your live URL and QR code

Copy the URL or scan the generated QR code to open the project on mobile instantly.

Supported file types

  • ZIP archives containing static project files
  • HTML, CSS, JavaScript assets inside your ZIP
  • Nested folders are supported if an index.html exists
  • Maximum upload size: 10MB per ZIP file

How QR sharing works

  • After upload, RunZip generates a unique live link for your project
  • A QR code is created from that link immediately
  • Anyone can scan it with a phone camera and open your project
  • No app install and no account required

FAQ

Is RunZip free?

Yes, RunZip is free to use.

Do I need an account?

No account or login is required.

How long are projects hosted?

Projects are automatically cleaned up after 24 hours.

What file size is supported?

Each upload supports ZIP files up to 10MB.

Does it work on mobile?

Yes. Open via direct link or QR code on phone and tablet.
